# 電力会社の乗り換え・太陽光・蓄電池——結局どの組み合わせが一番安い？

> **メタディスクリプション**: 電力会社の乗り換え、太陽光パネル、蓄電池。電気代を下げる手段はいくつかあるけれど、結局どの組み合わせがトータルで一番お得なのか？初期費用・補助金・売電収入を含めた10年シミュレーションで比較します。

---

電気代を下げる手段は大きく3つ。電力会社の乗り換え（初期費用ゼロ）、太陽光パネル（初期費用あり）、蓄電池（初期費用あり）。

「どこまでやるのがコスパいいのか？」——4つのパターンで10年間のトータルコストを比較します。

---

## 比較する4パターン

| パターン | 内容 | 初期費用の目安 |
|---------|------|-------------|
| A | 何もしない（大手電力のまま） | 0円 |
| B | 電力会社を乗り換えるだけ | 0円 |
| C | 乗り換え + 太陽光パネル | 約130万円 |
| D | 乗り換え + 太陽光 + 蓄電池 | 約290万円 |

---

## シミュレーションの前提条件

| 項目 | 設定値 |
|------|-------|
| 世帯 | 3〜4人家族（持ち家・戸建て） |
| 月間使用量 | 400kWh |
| 大手電力の電気代 | 約13,000円/月（従量三段階制） |
| 太陽光パネル | 5kW / 130万円（補助金適用前） |
| 蓄電池 | 7kWh / 160万円（補助金適用前） |
| 売電単価（FIT） | 16円/kWh |
| 補助金 | 太陽光: 約15万円 / 蓄電池: 約20万円（国＋自治体の目安） |

※金額はすべて2026年時点の目安です。地域や条件によって変わります。

---

## パターン別：10年間のトータルコスト

### パターンA: 何もしない

| 項目 | 金額 |
|------|------|
| 月々の電気代 | 約13,000円 |
| 年間の電気代 | 約156,000円 |
| **10年間の合計** | **約156万円** |

最もシンプルだが、最も高くつく。

### パターンB: 電力会社を乗り換える

| 項目 | 金額 |
|------|------|
| 初期費用 | 0円 |
| 月々の電気代 | 約11,500円（月約1,500円削減） |
| 年間の電気代 | 約138,000円 |
| **10年間の合計** | **約138万円** |
| パターンAとの差額 | **▲約18万円** |

月1,000〜2,000円の削減で10年18万円前後。手間に対するリターンが最も大きい。

### パターンC: 乗り換え + 太陽光パネル

| 項目 | 金額 |
|------|------|
| 初期費用 | 約130万円（補助金▲15万円 → 実質約115万円） |
| 月々の電気代 | 約6,500円（自家消費で大幅削減） |
| 売電収入 | 約3,000円/月 |
| 実質の月間コスト | 約3,500円 |
| 年間の実質コスト | 約42,000円 |
| **10年間の合計（初期費用込み）** | **約157万円** |
| パターンAとの差額 | **ほぼ同額（10年で回収）** |

約10年で初期費用を回収。11年目以降は年間約11万円がまるまる浮く計算。

**15年で見ると**：約99万円（パターンAより約57万円お得）

### パターンD: 乗り換え + 太陽光 + 蓄電池

| 項目 | 金額 |
|------|------|
| 初期費用 | 約290万円（補助金▲35万円 → 実質約255万円） |
| 月々の電気代 | 約3,000円（自家消費率アップ） |
| 売電収入 | 約1,500円/月（自家消費優先のため減少） |
| 実質の月間コスト | 約1,500円 |
| 年間の実質コスト | 約18,000円 |
| **10年間の合計（初期費用込み）** | **約273万円** |
| パターンAとの差額 | **＋約117万円** |

10年では回収しきれない。**20年で見ると**約291万円 vs パターンAの312万円。20年でようやく逆転する計算。

## 4パターンの比較まとめ

| パターン | 10年コスト | 15年コスト | 初期費用回収 |
|---------|-----------|-----------|------------|
| A: 何もしない | 156万円 | 234万円 | — |
| B: 乗り換えのみ | 138万円 | 207万円 | 即時（費用ゼロ） |
| C: 乗り換え+太陽光 | 157万円 | 99万円 | 約10年 |
| D: 乗り換え+太陽光+蓄電池 | 273万円 | 264万円 | 約18〜20年 |

## 結局どれがコスパいい？

### まずはパターンB（乗り換え）から

初期費用ゼロ、手続き10分、年間1〜2万円の削減。やらない理由がない。

### コスパ最強はパターンC（乗り換え + 太陽光）

10年で回収、11年目以降は年間10万円以上が浮く。パネル寿命は25〜30年なので長期ではトータルコストが大きく下がる。持ち家で屋根の条件に問題がなければ最もバランスの良い選択肢。

<!-- TODO: solar-panel-worth-it記事へのリンク -->

### パターンD（太陽光 + 蓄電池）は目的次第

**コスト削減だけなら蓄電池は割に合いにくい**のが現状。回収に20年近くかかる。

ただし、停電対策・卒FIT対応・電気代ゼロ志向など、金銭以外の目的があるなら検討の価値あり。蓄電池は年々値下がりしているため、太陽光を先に導入して数年後に後付けするのも賢い方法。

## 賃貸の場合

太陽光・蓄電池は設置できないが、電力会社の乗り換え（パターンB）は賃貸でも自由にできる。省エネの工夫やガスセット割と合わせれば、年間1〜2万円の削減は十分狙える。

---

## 見積もりは必ず3社以上から

同じ設備でも業者によって数十万円の差がつく。「今日契約すれば値引き」と急かす営業には警戒を。

## まとめ

| やること | 難易度 | 効果 | おすすめ度 |
|---------|-------|------|----------|
| 電力会社を乗り換える | 手続き10分 | 年1〜2万円削減 | 全員やるべき |
| 太陽光パネルを導入 | 見積もり〜工事 | 年10万円以上削減（回収後） | 持ち家ならおすすめ |
| 蓄電池を追加 | 見積もり〜工事 | 自家消費率アップ＋停電対策 | 目的次第 |

まずは乗り換え、次に太陽光、蓄電池は目的次第。この順番で検討するのが最もリスクが低い。
