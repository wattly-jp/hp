# 電力会社の乗り換え・太陽光・蓄電池——結局どの組み合わせが一番安い？

> **メタディスクリプション**: 電力会社の乗り換え、太陽光パネル、蓄電池。電気代を下げる手段はいくつかあるけれど、結局どの組み合わせがトータルで一番お得なのか？初期費用・補助金・売電収入を含めた10年シミュレーションで比較します。

電気代を下げたいとき、取れる手段は大きく3つあります。

1. **電力会社の乗り換え** — 今日からできる。初期費用ゼロ
2. **太陽光パネルの導入** — 自分で電気を作る。初期費用あり
3. **蓄電池の導入** — 作った電気を貯めて使う。初期費用あり

それぞれ単体の解説記事はよく見かけますが、「じゃあ全部やったらどうなるの？」「どこまでやるのがコスパいいの？」という疑問に答えてくれるものは意外と少ないです。

この記事では、4つのパターンで10年間のトータルコストを比較してみます。

---

## 比較する4パターン

| パターン | 内容 | 初期費用の目安 |
|---------|------|-------------|
| A | 何もしない（大手電力のまま） | 0円 |
| B | 電力会社を乗り換えるだけ | 0円 |
| C | 乗り換え + 太陽光パネル | 約130万円 |
| D | 乗り換え + 太陽光 + 蓄電池 | 約290万円 |

---

## シミュレーションの前提条件

現実に近い数字で比較するために、以下の条件を設定します。

| 項目 | 設定値 |
|------|-------|
| 世帯 | 3〜4人家族（持ち家・戸建て） |
| 月間使用量 | 400kWh |
| 大手電力の電気代 | 約13,000円/月（従量三段階制） |
| 太陽光パネル | 5kW / 130万円（補助金適用前） |
| 蓄電池 | 7kWh / 160万円（補助金適用前） |
| 売電単価（FIT） | 16円/kWh |
| 補助金 | 太陽光: 約15万円 / 蓄電池: 約20万円（国＋自治体の目安） |

※金額はすべて2026年時点の目安です。地域や条件によって変わります。

---

## パターン別：10年間のトータルコスト

### パターンA: 何もしない

大手電力会社の標準プランをそのまま使い続けるケースです。

| 項目 | 金額 |
|------|------|
| 月々の電気代 | 約13,000円 |
| 年間の電気代 | 約156,000円 |
| **10年間の合計** | **約156万円** |

最もシンプルですが、最も高くつきます。

### パターンB: 電力会社を乗り換える

初期費用ゼロ、手続き10分で始められる最もハードルの低い方法です。

| 項目 | 金額 |
|------|------|
| 初期費用 | 0円 |
| 月々の電気代 | 約11,500円（月約1,500円削減） |
| 年間の電気代 | 約138,000円 |
| **10年間の合計** | **約138万円** |
| パターンAとの差額 | **▲約18万円** |

使用量400kWhの家庭なら、新電力に乗り換えるだけで月1,000〜2,000円の削減が期待できます。10年で18万円前後。手間に対するリターンが最も大きい選択肢です。

### パターンC: 乗り換え + 太陽光パネル

電力会社の乗り換えに加えて、太陽光パネル（5kW）を導入するケースです。

| 項目 | 金額 |
|------|------|
| 初期費用 | 約130万円（補助金▲15万円 → 実質約115万円） |
| 月々の電気代 | 約6,500円（自家消費で大幅削減） |
| 売電収入 | 約3,000円/月 |
| 実質の月間コスト | 約3,500円 |
| 年間の実質コスト | 約42,000円 |
| **10年間の合計（初期費用込み）** | **約157万円** |
| パターンAとの差額 | **ほぼ同額（10年で回収）** |

10年間だとパターンAとほぼ同額。つまり約10年で初期費用を回収できる計算です。11年目以降は年間約11万円がまるまる浮くことになります。

**15年で見ると**：約99万円（パターンAより約57万円お得）

### パターンD: 乗り換え + 太陽光 + 蓄電池

太陽光パネルに蓄電池（7kWh）も追加するケースです。

| 項目 | 金額 |
|------|------|
| 初期費用 | 約290万円（補助金▲35万円 → 実質約255万円） |
| 月々の電気代 | 約3,000円（自家消費率アップ） |
| 売電収入 | 約1,500円/月（自家消費優先のため減少） |
| 実質の月間コスト | 約1,500円 |
| 年間の実質コスト | 約18,000円 |
| **10年間の合計（初期費用込み）** | **約273万円** |
| パターンAとの差額 | **＋約117万円** |

10年では初期費用を回収しきれません。蓄電池の分だけ回収期間が長くなります。

**20年で見ると**：約291万円 vs パターンAの312万円。20年でようやくパターンAより安くなる計算です。

## 結局どれがおすすめ？

### まずはパターンB（乗り換え）から

初期費用ゼロ、手続き10分。リスクなしで年間1〜2万円の削減ができます。ここをやらない理由がありません。まだ乗り換えていない方は、まずここから始めましょう。

### コスパ最強はパターンC（乗り換え + 太陽光）

10年で初期費用を回収し、11年目以降は年間10万円以上が浮きます。太陽光パネルの寿命は25〜30年なので、長期で見ればトータルコストは大きく下がります。

持ち家で屋根の向き・面積に問題がなければ、最もバランスの良い選択肢です。

### パターンD（太陽光 + 蓄電池）は目的次第

正直に言うと、**コスト削減だけが目的なら蓄電池の追加は割に合いにくい**のが現状です。蓄電池の価格がまだ高く、回収に20年近くかかります。

ただし、以下の目的があるなら検討する価値はあります。

- **停電対策**: 災害時に数日間の電力を確保できる安心感
- **卒FIT対応**: FIT期間（10年）終了後、売電単価が大幅に下がったときに自家消費率を上げられる
- **電気代ゼロを目指したい**: 月の電気代を限りなくゼロに近づけたい場合

蓄電池は年々価格が下がっているので、太陽光を先に導入して、数年後に蓄電池を後付けするのも賢い方法です。

## 賃貸の場合はどうする？

ここまでは持ち家を前提にしましたが、賃貸の場合は太陽光・蓄電池の導入はできません。

賃貸の方が取れる選択肢は以下の通りです。

- **電力会社の乗り換え**（パターンB）— 賃貸でも自由にできる
- **省エネの工夫** — LED照明、エアコンの使い方、待機電力カット
- **ガスとのセット割** — 同じ会社でまとめると割引になるケースも

賃貸でも電力会社の乗り換えだけで年間1〜2万円の削減は十分狙えます。

---

## 見積もりは必ず複数社から

太陽光パネルや蓄電池の導入を検討する場合、**必ず3社以上から見積もりを取りましょう**。同じ設備でも業者によって価格が数十万円違うことは珍しくありません。

また、訪問販売で「今日契約すれば値引き」と急かされるケースがありますが、焦って決める必要はまったくありません。相見積もりを取って冷静に比較してください。

## まとめ

| やること | 難易度 | 効果 | おすすめ度 |
|---------|-------|------|----------|
| 電力会社を乗り換える | 手続き10分 | 年1〜2万円削減 | 全員やるべき |
| 太陽光パネルを導入 | 見積もり〜工事 | 年10万円以上削減（回収後） | 持ち家ならおすすめ |
| 蓄電池を追加 | 見積もり〜工事 | 自家消費率アップ＋停電対策 | 目的次第 |

電気代を本気で下げたいなら、まずは一番簡単な電力会社の乗り換えから。そのうえで持ち家の方は太陽光パネルの導入を検討してみてください。

蓄電池は「コスト削減」だけでなく「安心」や「卒FIT対応」も含めて判断するのがポイントです。
