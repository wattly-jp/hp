# 記事テーマ一覧

テーマと切り口案のストック。**「エネチェンジにも書いてありそう」なテーマは除外**。

## 選定基準

全テーマは以下のいずれかを満たすこと:
- **問いを立てている**（「なぜ○○なのか」「○○は本当か」）
- **不都合な真実を突いている**（「○○の落とし穴」「○○が言わないこと」）
- **具体シチュエーションに特化**（「飲食店の」「マンション住まいの」「卒FIT後の」）
- **時事性がある**（ホルムズ海峡、料金改定、制度変更）
- **逆張り**（「○○しない方がいい人」「○○は意味がない」）

## 切り口テクニック

テーマを考えるとき、以下の手法を回す:

| 手法 | 例 |
|------|-----|
| **問いの逆転** | 「電気代が高い」と感じる基準は正しいのか |
| **不都合な真実** | 営業が言わないパワコン交換の落とし穴 |
| **他業界からの輸入** | 格安航空の欠航と新電力の倒産は同じ構造 |
| **具体シーンから入る** | 納車日にブレーカーが落ちた |
| **データファースト** | 119社が消えた、50万世帯分が捨てられている |
| **視点の転換** | 停電経験者と未経験者で評価が180度違う |
| **逆張り** | ガソリン車の方が合理的な人の条件 |
| **メタ視点** | 比較サイトは誰のために比較してるのか |

## 逆張り・メタ視点ストック

SNSでバズりやすい「え？」系テーマ:

### 電気代を気にしすぎて生活の質が下がる本末転倒
> 月300円の節約のためにエアコンを我慢して熱中症になった話
- 「節約する時間」を時給換算したら電気代より高かった
- 健康オタクが逆に不健康になる現象と同じ構造

### 電力会社の中の人が自宅で使ってる電力会社
> 業界人の本音。取材・アンケート系で独自性が出やすい
- 車のディーラーが自分では別メーカーに乗ってるあの感じ

## 内部リンクルール

- **ピラー → クラスター**: ピラー記事から全クラスター記事にリンク
- **クラスター → ピラー**: 各クラスター記事からピラーに戻るリンク
- **クラスター → クラスター**: 同一クラスター内の関連記事同士をリンク
- **クロスクラスター**: 自然な文脈があれば他クラスターの記事にもリンク
- **記事公開時に既存記事の内部リンクも更新する**（新記事への逆リンク追加）
