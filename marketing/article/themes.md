# 記事テーマ一覧

テーマと切り口案のストック。記事化したいものに `- [x]` をつける → まとめてClaude に渡すと一気にdraftを生成する。

**使い方:**
1. 記事化したいテーマの先頭チェックボックスを `- [x]` に変える
2. 「チェックしたやつdraft化して」とClaude に伝える
3. Claude がチェック済みのテーマを全てdraft化 → draft化したthemeは削除

切り口は「具体（入口）→ 抽象（本質）」の順で設計する。具体と抽象はなるべく遠いところを行き来する。比喩・比較もなるべく遠い業界・領域から持ち込む。**タイトル自体に「なぜその概念が？」という引っかかりを持たせる。**

## 構造・視座系

- [ ] ### 電球が切れたとき、人は初めて「天井」の存在を思い出す
> 電力インフラは「ないと困る」が「あっても気づかない」——意識に上らないものが社会を支えている構造
- 存在を意識されるインフラほど、実は機能が低下している——水道・道路・電力の「気づかれなさ」と信頼性の関係（他業界：インフラ工学）
- 日本の年間停電時間は世界最短水準（数分〜十数分）。この「気づかれなさ」に年間いくら払っているか
- 医療でも同じ：名医は患者が「気づかない病気」を予防している。問題が起きないことの価値は見えない
- **具体（入口）**: 能登地震で初めて「電気がある生活」の構造を認識した人たち
- **抽象（本質）**: 信頼性への対価は、信頼性が壊れた時しか可視化されない

- [ ] ### 翻訳家は、元の文章を「壊す」ことで意味を伝える
> 電気料金の「わかりにくさ」は設計ミスではなく設計の結果——複雑さを誰が利益にしているかを問う
- 翻訳理論：直訳は意味を殺す。意味を届けるために、元の構造を一度壊す必要がある（他業界：言語学）
- 電気料金の構成要素（基本料金・電力量料金・燃料費調整額・再エネ賦課金）——複雑な計算式が「比較をやめさせる」機能を果たしている
- 保険商品・クレジットカード手数料・携帯電話プラン——「わかりにくい」料金体系で利益を得る構造は共通している
- **具体（入口）**: 請求書を見ても「なぜこの金額なのか」が説明できない人の割合
- **抽象（本質）**: 複雑さは「消費者への説明責任の回避」として機能することがある

## エネルギー・社会系

- [ ] ### 火山の麓に住む人は、火山を恐れていない
> リスクに慣れ親しんだ人の感覚は狂う——「停電が怖くない」「電力会社が倒産しても大丈夫」という感覚の危うさ
- 火山リスク研究：火山の近くに住む人ほど噴火確率を低く見積もる——日常化がリスク感覚を麻痺させる（他業界：災害心理学）
- 2022年の新電力撤退時、「うちは大丈夫だろう」と思っていた顧客の体験
- 「今まで何もなかった」は「これからも何もない」の根拠にならない——経験と確率の乖離
- **具体（入口）**: 「倒産するような会社じゃないと思ってた」という撤退新電力の元顧客
- **抽象（本質）**: 慣れは「リスクの消滅」ではなく「リスク感覚の麻痺」

- [ ] ### 夜中に電気が余っても、誰も喜ばない
> 電力は「作りすぎると捨てるしかない」——需給バランスの非対称性と、その歪みが価格に与える影響
- 牛乳廃棄の構造：需要を超えた供給は捨てるしかない。電気も同じ——ただし在庫ゼロで翌朝リセットされる（他業界：農業・食品）
- 深夜の風力・太陽光の余剰電力が「マイナス価格」になる現象（欧州で実例多数）
- 「電気を貯める」技術（蓄電池・揚水発電・水素変換）がなぜ急ぎで必要なのか
- **具体（入口）**: 九州電力が太陽光の余剰で「出力制御」を命じた日、パネルオーナーが受けた損失
- **抽象（本質）**: 「作りすぎた電気」には価値がない——電気の価値は「必要な時に届く」ことにある

## 家計・節約系

- [ ] ### 賃貸に住んでいる人の電気代は、大家が決めている
> 入居者が電力会社を選べない構造——建物の断熱・設備・契約形態が「固定コスト」として押しつけられる
- 小作農制度：土地の収益構造は地主が決め、農民はそれに従う——賃貸の断熱問題は「誰が投資し誰が得するか」の構造問題（他業界：農業経済・封建制）
- 断熱等級の低い物件に住む入居者は、年間数万円分のエネルギーを「暖房で壁から外に捨てている」
- 省エネ改修をする動機があるのは大家（資産価値向上）だが、電気代を払うのは入居者——インセンティブの分離
- **具体（入口）**: 断熱が悪い1Kに住んで月2万円の電気代を払い続けていた入居者
- **抽象（本質）**: 賃貸の電気代は「使い方」ではなく「建物の構造」で決まる部分が大きい

- [ ] ### 「電気代を節約する」という問いの立て方が、すでに間違っている
> 「減らす」より「ずらす」方が合理的——ピーク時間と料金単価の関係
- 渋滞研究：道路の渋滞は「総量の問題」より「集中の問題」——全員が同じ時間に移動するから詰まる（他業界：交通工学）
- 電気代の「値下がり」は使用量削減より時間帯シフトで起きやすい——深夜電力・オフピーク割引の仕組み
- エアコンの設定温度より「いつ使うか」の方が電気代への影響が大きいケース
- **具体（入口）**: 節電を頑張っても変わらなかった電気代が、洗濯機を深夜に回すだけで下がった家庭
- **抽象（本質）**: エネルギーコストは「量」より「タイミング」で決まることがある

## 太陽光・蓄電池系

- [ ] ### 20年ローンを組んで買った家が、15年で建て替えになることがある
> 太陽光パネルの「元を取る」計算に、出口コストが含まれていない問題
- 建築の「建て替えリスク」：ローン返済中に解体を迫られた建物の話。取得コストと廃棄コストを分けて考えない設計の罠（他業界：建築・不動産）
- 太陽光パネルの撤去・廃棄費用（2040年代に数百万台が廃棄期を迎える）が試算に含まれていないケースがほとんど
- FIT終了後の売電価格下落——「20年で回収」の前提が崩れると、計算がどう変わるか
- **具体（入口）**: 「得だと思って設置した」パネルの廃棄費用を初めて知った家庭
- **抽象（本質）**: 投資の採算計算に「出口コスト」が入っていないと、損益分岐点が大きくずれる

- [ ] ### 保険会社が嫌いなリスクが、一番大きいリスクだ
> 蓄電池・太陽光の「10年保証」に、保険会社が損害保険をつけたがらない理由
- 保険数理：保険会社がリスクを引き受けない領域は、損失の予測ができない領域——原発・巨大地震と同じカテゴリ（他業界：保険・アクチュアリー）
- 太陽光パネルの火災リスク・廃棄物リスクは「確率はわかるが規模が読めない」部分がある
- メーカー保証は「会社が存在することが前提」——撤退・倒産後の保証の実効性
- **具体（入口）**: 保証書があったのに、メーカーが廃業していて修理できなかったケース
- **抽象（本質）**: 保証の価値は「書類の内容」ではなく「保証を実行できる主体が存在するか」で決まる

## 法人系

- [ ] ### 工場の電気代は、「使った量」より「一瞬の最大値」で決まる
> デマンド（最大需要電力）の仕組みを知らないまま払い続けている法人の構造
- マラソンの「スタートダッシュ問題」：ペースは平均ではなく最初の1kmで全体タイムが決まる——電力の基本料金はピーク1点で決まる（他業界：スポーツ科学）
- 高圧電力の基本料金は月間最大需要電力（デマンド）で計算される——月の0.01%の「一瞬」が1ヶ月分の固定費を決める
- デマンドを1kW下げるだけで年間数万〜数十万円変わるケースがある
- **具体（入口）**: 電力消費を削減したのに基本料金が変わらなかった工場の「なぜ」
- **抽象（本質）**: 電力コストは「平均の消費量」ではなく「最大の瞬間」によって支配される

- [ ] ### 電気を「使わない」ことが、売り物になる日が来た
> ディマンドレスポンス（DR）——電力逼迫時に電気を切ることで報酬を得る仕組み
- 株式の「空売り」：持っていないものを売ることで収益を得る——「使わないことの取引」という発想の輸入（他業界：金融）
- 電力逼迫時に工場の電力消費を下げると、電力会社から報酬が支払われるDR制度の仕組み
- 「電力を節約する」は義務のように語られていたが、「節約を売る」に変わるとビジネスになる
- **具体（入口）**: 年間数百万円のDR報酬を受け取っている工場の話
- **抽象（本質）**: コストセンターだったエネルギー管理が、プロフィットセンターに変わる設計がある

- [ ] ### 中小企業が大企業より高い電気代を払う理由は、公平性とは無関係だ
> 電力契約の「50kWの壁」——規模によって単価が変わる構造と、それを知らない経営者
- 高速道路の料金体系：大型トラックと乗用車で重量あたりの料金が違う——インフラは量によって単価が変わる（他業界：物流・交通）
- 低圧50kW未満と高圧50kW以上で電気の単価が大きく変わる——しかし境界線を意識していない中小企業がほとんど
- 契約を変えるだけで単価が下がるが、変えるために設備投資が必要——「知らないコスト」と「知らない節約」が共存している
- **具体（入口）**: 隣の工場より電気代が割高なことを、契約変更で初めて理解した中小企業のオーナー
- **抽象（本質）**: 不公平な価格体系は「差別」ではなく「構造の無知」によって維持される

## 技術の仕組み・深掘り系

- [ ] ### 核融合は「制御できない爆発」を飼いならす技術だ
> 太陽のエネルギーを地上で再現する核融合炉——その仕組みと、商用化がなぜ「常に30年先」と言われ続けるか
- 核融合の物理：水素の同位体（重水素・三重水素）をプラズマ状態にして1億℃で衝突させると、ヘリウムになって余剰エネルギーが放出される——これが核融合反応
- 「制御」の難しさ：プラズマは不安定で、強力な磁場（トカマク型）やレーザー（慣性閉じ込め型）でしか容器に保持できない。制御を失えば反応は即座に止まる（爆発はしないが、エネルギーを取り出せない）
- 核分裂との根本的な違い：核分裂（現在の原発）は重いウラン原子を割る反応で、放射性廃棄物が残る。核融合は軽い水素を合わせる反応で、長寿命の放射性廃棄物が少ない
- ITER（国際熱核融合実験炉）：フランスに建設中の巨大実験炉。2035年以降の運転開始予定で、Q=10（入力エネルギーの10倍を出力）を目標にしている
- 民間参入の加速：Commonwealth Fusion Systems・Helion Energy・TAE Technologiesなど、2020年代に入って民間核融合スタートアップが急増。「常に30年先」が変わり始めているという見方も
- **具体（入口）**: 「核融合発電が実現したら電気代はどうなるか」という問いを持つ読者
- **抽象（本質）**: 核融合は「無限のエネルギー」という夢ではなく、「制御技術と材料科学の限界への挑戦」として理解すると、商用化の困難さと可能性の両方が見える

- [ ] ### EVの「充電」は、ガソリン給油とは別の物理で動いている
> 電気自動車の充電インフラの仕組み——なぜ急速充電はバッテリーを傷めるのか、なぜ自宅充電が最も合理的なのか
- 充電の3レベル：普通充電（200V/3kW）・急速充電（50kW〜）・超急速充電（150kW〜350kW）——電流・電圧・化学反応速度のバランスで充電時間と劣化が決まる
- リチウムイオン電池の仕組み：リチウムイオンが正極（LFP・NMCなど）と負極（黒鉛）の間を行き来することで充電・放電する。充電速度を上げすぎるとリチウムが析出して劣化が加速する（リチウムメッキ現象）
- 急速充電が速い理由と代償：高電流を流すことで短時間で充電できるが、発熱とイオン移動の乱れが積み重なり、バッテリー容量が長期的に低下する。多くのEVメーカーが「急速充電は週1〜2回以下を推奨」と言う
- 自宅充電が最も合理的な理由：夜間の低電力料金帯を使い、低速（3kW）でゆっくり充電できる。バッテリーへの負荷が最小で、電気代も最も安い
- V2H（Vehicle to Home）の可能性：EVのバッテリーを家庭の蓄電池として使う技術。太陽光と組み合わせると、日中に発電した電気をEVに溜め、夜間に家庭で使える
- 充電インフラの「距離の壁」：高速道路のSAや道の駅の急速充電器は設置コストが高く、採算が取りにくい。民間事業者が参入しにくい構造が、地方の充電インフラ不足の根本原因
- **具体（入口）**: EVに乗り始めたが、急速充電を使いすぎてバッテリーが劣化していると知った話
- **抽象（本質）**: EVの充電を「液体を入れる」のではなく「化学反応を管理する」と捉えると、なぜ「充電の仕方」が性能に直結するかが理解できる

## 優先順位の考え方

1. **構造・視座系を柱にする**: SNSシェア・被リンク狙い。SEO直結より「Wattlyといえばこの切り口」というブランド形成
2. **エネルギー・社会系は時事性で伸ばす**: 能登地震・出力制御など具体的な事件と紐づける
3. **家計系はロングテール流入の入り口**: 「電気代 下がらない」「節電 意味ない」系の検索に刺さる構成
4. **法人系はCV直結**: 検索ボリュームは小さいが高単価アフィリエイトに繋がる。家計系の内部リンク先として後から書く
